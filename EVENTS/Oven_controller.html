Title: Reflow Oven Controller

Date: Jan- Feb, 2019

Location: UBC, Vancouver

Team Members:

Jian Gao, Jacky Yu, JoJo Windsor-Lewis
Clarence Adrian, Matthew Lee, Hao Luo

Supervisor: Dr. Jesús Calviño-Fraga

Electrical Engineering Design Studio Project I

The objective of the project entails the software and hardware design of an oven controller accurate enough to conduct reflow soldering, a process commonly used to temporarily attach electrical components on a printed circuit board (PCB) by the use of solder paste. Design stages include the building, programming, and testing of the prototype, along with the implementation of extra functionalities, with an allocated work time of a few weeks. Below are the further requirements for the project:

  1. Programmed in assembly language
  2. Able to measure temperatures between 25oC and 240oC using a K-Type Thermocouple
  3. Able to operate a standard off-the-shelve 1500W toaster using a Solid State Relay (SSR)
  4. Basic User Interface and Feedback Functionality
  5. Error/exception handling: automatic cycle termination
  6. Temperature validation data to show ±3°C temperature accuracy for the given range

Additional learning objectives:

  1. Acquire knowledge of assembling PCBs with SMT components by reflow soldering
  2. Understand the components of a reflow oven controller
  3. Implement a thermocouple as a method of measuring temperature
  4. Control and load AC currents throughout different electrical components using a Solid State Relay
